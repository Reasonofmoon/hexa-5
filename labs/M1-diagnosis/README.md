## 🔵 Professional: AI 진단 프롬프트 10개

### 프롬프트 1: 공사 기간 예측 최적화
```
당신은 건설 프로젝트 관리 AI 전문가입니다.
[아파트 건설 현장]의 공사 데이터를 분석하여 기간 예측 모델을 개발해주세요.
- 현재 공사 지연율: [22%]
- 예측 정확도: [68%]
- 목표: 지연율 10% 이하, 예측 정확도 85% 이상
과거 공사 데이터와 기상 정보를 반영한 공사 기간 예측 시스템을 구축해주세요.
```

### 프롬프트 2: 자재 수요 예측 및 재고 관리
```
당신은 건설 자재 관리 AI 전문가입니다.
[건설 현장]의 자재 사용 데이터를 분석하여 최적화 방안을 제시해주세요.
- 현재 자재 낭비율: [15%]
- 재고 부족 발생률: [18%]
- 목표: 낭비율 8% 이하, 부족 발생률 5% 이하
공사 스케줄 기반 자재 수요 예측과 자동 발주 시스템을 구축해주세요.
```

### 프롬프트 3: 안전 관리 및 위험 예측
```
당신은 건설 안전 AI 전문가입니다.
[고층 빌딩 공사 현장]의 안전 데이터를 분석하여 위험 예측 시스템을 설계해주세요.
- 현재 안전사고 발생률: [0.8%]
- 위험 감지율: [45%]
- 목표: 사고 발생률 0.3% 이하, 위험 감지율 80% 이상
실시간 영상 분석과 IoT 센서 데이터를 활용한 안전 관리 시스템을 개발해주세요.
```

### 프롬프트 4: 품질 관리 자동화
```
당신은 건설 품질 AI 전문가입니다.
[주택 단지]의 품질 검수 데이터를 분석하여 자동화 방안을 제시해주세요.
- 현재 결함 발견율: [75%]
- 검수 시간: [동당 2시간]
- 목표: 발견률 90% 이상, 검수 시간 30분 이내
드론 촬영과 이미지 분석 기반 품질 검수 자동화 시스템을 구축해주세요.
```

### 프롬프트 5: 비용 최적화 및 예측
```
당신은 건설 비용 AI 전문가입니다.
[상업용 건물]의 비용 데이터를 분석하여 최적화 전략을 수립해주세요.
- 현재 예산 초과율: [18%]
- 비용 예측 정확도: [72%]
- 목표: 예산 초과율 8% 이하, 예측 정확도 90% 이상
재료비, 인건비, 설비비를 통합한 비용 예측 모델을 개발해주세요.
```

### 프롬프트 6: 부동산 가격 예측
```
당신은 부동산 AI 전문가입니다.
[서울 아파트]의 가격 데이터를 분석하여 예측 모델을 구축해주세요.
- 현재 가격 예측 정확도: [78%]
- 시장 변동 대응: [3개월 지연]
- 목표: 예측 정확도 85% 이상, 실시간 대응
위치, 시설, 교통, 학군 등 다차원 요인을 반영한 가격 모델을 개발해주세요.
```

### 프롬프트 7: 임대 수익 최적화
```
당신은 부동산 임대 AI 전문가입니다.
[상업용 임대 건물]의 임대 데이터를 분석하여 수익 최적화 전략을 제시해주세요.
- 현재 공실률: [12%]
- 평균 임대 수익률: [5.2%]
- 목표: 공실률 5% 이하, 수익률 7% 이상
시장 동향과 임대인 프로필 기반 임대 가격 책정 모델을 구축해주세요.
```

### 프롬프트 8: 건물 유지관리 예측
```
당신은 부동산 유지관리 AI 전문가입니다.
[오피스 빌딩]의 유지관리 데이터를 분석하여 예측 시스템을 설계해주세요.
- 현재 예방 정비율: [35%]
- 긴급 수리 발생률: [8%]
- 목표: 예방 정비율 70% 이상, 긴급 수리 3% 이하
IoT 센서와 머신러닝 기반 설비 상태 예측 시스템을 개발해주세요.
```

### 프롬프트 9: 부동산 투자 분석
```
당신은 부동산 투자 AI 전문가입니다.
[다세대 주택]의 투자 데이터를 분석하여 수익성 모델을 구축해주세요.
- 현재 투자 수익률: [연 6.5%]
- 위험 평가 정확도: [70%]
- 목표: 수익률 연 8% 이상, 위험 평가 정확도 85% 이상
지역 경제, 인구 변동, 정책 변화를 반영한 투자 분석 모델을 개발해주세요.
```

### 프롬프트 10: 스마트홈/빌딩 통합
```
당신은 스마트 부동산 AI 전문가입니다.
[신규 아파트 단지]의 스마트 시스템을 설계해주세요.
- 현재 에너지 효율: [65%]
- 거주자 만족도: [72점/100점]
- 목표: 에너지 효율 85% 이상, 만족도 85점 이상
IoT 통제, 에너지 관리, 보안 시스템을 결합한 스마트홈 솔루션을 구축해주세요.
```
